# 这是对SSG的一次尝试，是基于python的MkDocs包

### 文件夹*InForGH*(*Introduction For GreenHands*)下的*Intro_For_MkDocs*文件夹就是**完整**的网页文件夹，其中*index.html*文件就是web网页文件,如果想要看到web形式的成品，建议将这个*Intro_For_MkDocs*完整文件夹clone到本地然后打开*index.html*。

### *docs文件夹*和*mkdocs.yml*是一组，是在编辑过程中存在的两个内容，理论上捆绑在一起，在*docs文件夹*中的*index.md*文件就是网页的骨干内容，和网站内容上基本没有区别，可以作为预览，看最终的web成品只需要*Intro_For_MkDocs*文件夹。
