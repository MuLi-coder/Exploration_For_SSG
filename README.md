# 这是对SSG的一次尝试，是基于python的MkDocs包

### 文件夹InForGH(Introduction For GreenHands)下的Intro_For_MkDocs文件夹就是完整的网页文件夹，其中index.html文件就是web网页文件,如果想要看到web形式的成品，建议将这个Intro_For_MkDocs完整文件夹clone到本地然后打开index.html。

### docs文件夹和mkdocs.yml是一组，是在编辑过程中存在的两个内容，理论上捆绑在一起，在docs文件夹中的index.md文件就是网页的骨干内容，和网站内容上基本没有区别，可以作为预览，看最终的web成品只需要InForGH文件夹。
